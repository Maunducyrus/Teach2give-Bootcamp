1. What is Git?

Git is a distributed version control system that tracks versions of files. It is often used to control source code by programmers who are developing software collaboratively.

A version control system (VCS) is a tool that helps developers manage changes to source code over time, in this blog you will be going through Git and GitHub.

Other VCS examples:-

  1. SVN (subversion)
  2. Mercurial
  3. Perforce
