1. What is Git?

Git is a distributed version control system that tracks versions of files. It is often used to control source code by programmers who are developing software collaboratively.

A version control system (VCS) is a tool that helps developers manage changes to source code over time, in this blog you will be going through Git and GitHub.

Other VCS examples:-

  1. SVN (subversion)
  2. Mercurial
  3. Perforce

4. Initializing a Git Repository

A Git repository is a database that stores all versions of a project and its history, enabling developers to track changes, collaborate, and revert to previous versions

5. Adding files to the staging area

The staging area is a file, generally contained in your Git directory, that stores information about what will go into your next commit.

Tracked filesThese are files that have been added to the Git repository using the git add command and committed with git commit.
